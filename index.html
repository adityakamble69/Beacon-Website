// FAQ

Questions, answered

Everything you'd ask before connecting a real repo.

Does Beacon see our proprietary code?+
Yes — Beacon indexes whatever repo and docs folder you connect. Everything is encrypted per-tenant at rest and in transit, and it's never used to train any model. See the Security page for the full breakdown.
How long does setup actually take?+
Most teams are answering real questions in the same session — connect a repo, point it at a docs folder, and indexing starts immediately. No export step, no implementation call.
What happens when we push new code?+
A push triggers re-indexing automatically, scoped to just the files that changed — not the whole repo — so answers stay current within minutes.
Can we self-host Beacon?+
Self-hosting is available on the Enterprise tier, alongside SSO and audit logs. Talk to us for details on deployment options.
What if Beacon doesn't know the answer?+
It says so instead of guessing. Beacon only answers from a grounded, cited source — if it can't find one, it tells the developer rather than filling the gap.
Do you offer a free plan?+
Yes — the Starter tier is free forever, with one assistant and 500 queries a month, no credit card required.
